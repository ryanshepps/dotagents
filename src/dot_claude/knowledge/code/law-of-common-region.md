# Law of Common Region

> Elements inside a shared boundary read as a group — use containers, cards, and backgrounds to cluster related content.

## Key Takeaways

- A border, shared background, or container signals "these belong together" faster than spacing alone.
- Common region beats proximity when items must be close to unrelated content — the boundary overrides raw distance cues.
- Do not over-nest containers. Each border adds visual weight; use them only where grouping adds clarity.
