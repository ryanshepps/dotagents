# Email Format

> Email-specific rules — subject states the ask, greeting matches relationship, body leads with the lede, sign-off matches register. Skip ceremony when the relationship allows.

## Subject Line

The subject is the email's first sentence. Treat it as the lede.

- **Action emails:** start with a verb or a noun phrase that names the ask. `Approval needed: Q2 budget`, `Reviewing your draft`, `Decision: deprecate v1 auth`.
- **FYI emails:** start with a topic tag. `[FYI] Migration timeline shifting`, `[Update] Vendor contract signed`.
- **Threads:** keep the subject if you're continuing the same topic. Edit it (or start a new thread) if the topic shifted.

⊥ vague subjects: "Quick question", "Hey", "Following up", "One more thing". They tell the recipient nothing about whether to open now or later.

## Greeting — match the relationship

| Relationship | Greeting |
|--------------|----------|
| First contact, formal | `Dear <First Last>,` |
| First contact, modern | `Hi <First>,` |
| Recurring colleague | `Hi <First>,` or `Hey <First>,` |
| Close coworker | `<First>,` or no greeting |
| Group, formal | `Hi all,` `Hi team,` |
| Group, casual | `Hey folks,` `Team,` |

⊥ over-formal greetings to people you Slack daily — feels off, suggests bad news.
⊥ "To whom it may concern" unless you genuinely don't know who'll read it.

## Body — lede first

Apply [[lead-with-the-news]]. First sentence states the ask, decision, or news. Context comes after.

For multi-point emails, use a numbered list or bold sub-headers. Reader should be able to skim and reply to each point separately.

For long emails (>250 words), open with a TL;DR sentence, then `## Details` for the rest.

## Sign-off — match the register

| Register | Sign-off |
|----------|----------|
| Formal external | `Best regards,` / `Sincerely,` |
| Standard professional | `Best,` / `Thanks,` |
| Internal recurring | `Thanks,` / `— <name>` / just first name |
| Close coworker | first name only or nothing |

The sign-off should match the greeting in formality. `Dear Dr. Lee,` paired with `— ryan` reads off; so does `Hey Sam,` paired with `Sincerely,`.

## When to break ceremony

For ongoing fast threads with a tight team, drop greeting and sign-off entirely. The thread context tells everyone who's writing. Re-add ceremony when:
- A new person joins the thread
- The topic shifts to something formal (offer letter, exit, escalation)
- You're forwarding to leadership or external parties

## Tone check on email

Common smells:
- Subject doesn't name the ask → reader has to open to triage.
- Buried ask several paragraphs in → many readers stop early.
- Mismatched greeting/sign-off register → reads as nervous or insincere.
- Hedging stack ("I just wanted to maybe quickly ask if perhaps…") → name the ask directly.
- Apologizing for length → cut the email instead of apologizing.
