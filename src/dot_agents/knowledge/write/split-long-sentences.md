# Split Long Sentences

> Break long sentences into shorter standalone sentences. Same meaning, less cognitive load. Periods are free.

## Why

A sentence is a single unit of attention. The longer it runs, the more the reader has to hold in working memory before the period releases them. Compound sentences with three commas, a semicolon, and a parenthetical aside don't reward the patience — they tax it.

Splitting almost never loses meaning. It usually exposes that two of the three clauses were doing the same job, and one can be cut entirely.

## Pattern

Look for sentences with:

- Multiple independent clauses joined by `and`, `but`, `which`, `because`
- A semicolon
- More than ~25 words
- A parenthetical that interrupts the spine

Try replacing the joiner with a period. Read both versions aloud. If the split version reads cleanly, keep it.

## Examples

**Long:**
> The migration was risky because the legacy schema had implicit foreign keys that weren't enforced at the database level, which meant we had to write a custom validator to catch orphan rows before the cutover, and even then we found three more during the dry run.

**Split:**
> The migration was risky. The legacy schema had implicit foreign keys, unenforced at the database level. We wrote a custom validator to catch orphan rows before the cutover. Even then, we found three more during the dry run.

Same content, four sentences instead of one. Each sentence lands one fact. The reader can stop, breathe, and continue.

## When long sentences earn it

- **Cumulative effect** — a long sentence that piles detail to overwhelm or carry the reader is a deliberate choice. Read aloud to test whether yours is doing this work.
- **Legal or specification prose** where precision requires nested conditionals.
- **Voice** — some authors write long for cadence. If the rhythm works, leave it.

Default is split. Keep long when it's earning.

## In critique mode

Flag any sentence over ~30 words. Don't rewrite — point at the joiner and say "try a period here." The author chooses.
