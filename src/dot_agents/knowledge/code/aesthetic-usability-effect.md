# Aesthetic-Usability Effect

> Users perceive aesthetically pleasing designs as more usable, even when underlying usability is identical.

## Key Takeaways

- Polished visuals buy goodwill. Users tolerate minor friction in a beautiful interface and report it as more usable than a functionally identical ugly one.
- The flip side: aesthetics can mask real usability problems during testing. Users may not report issues they experience in a product that looks good.
- Do not treat visual polish as decoration — it is a usability lever. But pair it with real testing so the halo effect does not hide defects.
