# Goal-Gradient Effect

> Motivation rises the closer users get to a goal — show progress to pull them across the finish line.

## Key Takeaways

- Progress bars, step indicators, and completion percentages boost follow-through. Users push harder when the end is visible.
- "Endowed progress" helps too — starting a user at 20% on a 10-stamp card beats starting at 0% on an 8-stamp card, even when effort is identical.
- For long flows, break into phases so each segment has a visible end within reach.
