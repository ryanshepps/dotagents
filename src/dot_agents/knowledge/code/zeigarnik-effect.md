# Zeigarnik Effect

> Users remember incomplete tasks better than completed ones — use open loops to draw them back.

## Key Takeaways

- Unfinished tasks stay mentally active. Checklists, "3 of 5 steps done," and saved drafts exploit this to pull users back.
- Pair with the goal-gradient effect: visible remaining steps + visible progress creates strong pull toward completion.
- Do not weaponize it. Permanent "1 step left!" nags that never resolve become spam and erode trust.
