# Serial Position Effect

> Users best remember the first and last items in a list — put the most important items at those positions.

## Key Takeaways

- Primacy and recency dominate recall. Middle items fade fastest.
- In nav bars, place core actions at the edges (Home on the far left, Profile/Cart on the far right). Same logic for menus and settings.
- When order matters for decisions (e.g. pricing tiers), the first option anchors and the last leaves a fresh impression — design the middle as the "recommended."
