# Law of Proximity

> Elements close together read as related — use spacing as the primary grouping tool.

## Key Takeaways

- Whitespace is the cheapest grouping signal. Tight cluster = related; wide gap = separate.
- Label-to-input distance matters: a label closer to the wrong field confuses users even if styling is correct.
- Proximity applies to all axes — vertical grouping in lists, horizontal in toolbars, radial around focal points.
