# Cognitive Load

> The mental effort required to use an interface — minimize it, because users have a finite budget per task.

## Key Takeaways

- Three types: intrinsic (task itself), extraneous (poor design), germane (learning). You can only cut extraneous — that is where UX work pays off.
- Reduce load by removing steps, reusing familiar patterns, showing only relevant information, and offloading memory to the interface (recognition over recall).
- High cognitive load manifests as errors, abandonment, and support tickets — not complaints. Watch behavior, not feedback.
