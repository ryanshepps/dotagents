# Von Restorff Effect

> An item that looks different from its neighbors gets remembered — use contrast sparingly to highlight what matters.

## Key Takeaways

- Also called the isolation effect. A single bold CTA among muted secondary buttons is instantly scannable.
- Contrast is a finite resource. Highlighting everything highlights nothing — pick the one thing worth the emphasis per view.
- Do not rely on color alone. Accessibility requires pairing contrast with shape, weight, or icon so the distinction survives color-blindness and low-contrast displays.
