# Chunking

> Group related information into meaningful units so users can process and recall it more easily.

## Key Takeaways

- Working memory handles chunks, not characters. `4155550123` is one blob; `(415) 555-0123` is three chunks and far easier to hold.
- Chunk content visually (whitespace, headings, cards) and semantically (group related fields together).
- Chunk size matters too — keep each group small enough to scan at a glance.
