# Choice Overload

> Too many options paralyze users — decision quality and satisfaction drop as choice count rises.

## Key Takeaways

- Large option sets increase time-to-decide and abandonment rate. Users often pick nothing rather than pick wrong.
- Mitigate with smart defaults, categorization, progressive disclosure, and filters that narrow the visible set to a handful at a time.
- Highlight a recommended option to reduce the cognitive cost of evaluation.
