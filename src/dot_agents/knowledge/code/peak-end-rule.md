# Peak-End Rule

> Users judge experiences by their peak moment and their end, not the average — invest heavily in both.

## Key Takeaways

- A great peak and strong ending can redeem a mediocre middle. A bad ending poisons an otherwise smooth experience.
- Celebrate completion — confirmation screens, success animations, thank-yous — this is the "end" memory users take away.
- Handle errors with care. A gracious failure mode often beats a forgettable success at retaining users.
