# Selective Attention

> Users focus on a small subset of the screen and ignore the rest — design for where attention lands, not where content lives.

## Key Takeaways

- Banner blindness is real. Content that looks like an ad (or any ignored pattern) gets filtered out, even if the user "sees" it.
- Place critical info in the user's task path — not in peripheral regions, not in styles users have learned to dismiss.
- Novelty and motion grab attention but fatigue fast. Reserve them for moments that genuinely matter.
